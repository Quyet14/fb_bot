👤 Thêm tài khoản Facebook

Nhập thông tin đăng nhập Facebook. Bot sẽ dùng tài khoản này để lấy danh sách fanpage bạn quản lý. Dữ liệu chỉ lưu trên server của bạn.

⚡ User Access Token (tùy chọn — để đăng bài nhanh hơn)
Để lấy token: vào Graph API Explorer → Generate Token → Cấp quyền pages_manage_posts + pages_read_engagement → Paste vào đây.
Nếu không nhập, bot sẽ dùng Selenium (chậm hơn).
